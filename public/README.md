# Public Assets

Place your images in this directory:

## Required Images

1. **hero-main.png** - Hero section background image (recommended: 1920x1080 or larger)
2. **recruitment-image.jpg** - Recruitment card hover background image
3. **alumni-image.jpg** - Alumni card hover background image
4. **instagram-1.jpg** through **instagram-6.jpg** - Instagram feed images (recommended: square format, 800x800 or larger)

## Image Optimization Tips

- Use WebP format for better performance (convert JPG/PNG to WebP)
- Compress images before adding them
- Ensure hero-main.png is optimized for web (consider using a compressed version)
